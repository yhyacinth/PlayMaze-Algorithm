# PlayMaze!

Not new, C++ maze generator.

* Maze generation with Prim's algorthm
* 2D Maze, 3D Maze
* Masking
* OpenGL
* Maze Canvas Model-To-View, View-To-Model
* Maze Solver (Set a Start and End Point)
